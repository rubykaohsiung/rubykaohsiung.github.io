Ruby on Rails 指南 (v6.1.4)

這些是基於 v6.1.4 的 Rails 7.0 新指南。 這些指南旨在讓您立即使用 Rails 提高工作效率，並幫助您了解所有部分是如何組合在一起的。

早期版本的指南： Rails 6.1， Rails 6.0、 Rails 5.2， Rails 5.1， Rails 5.0， Rails 4.2， Rails 4.1、 Rails 4.0、 Rails 3.2， Rails 3.1、 Rails 3.0，以及 Rails 2.3。

Rails 指南也可用於 Kindle。

從這裡開始

Rails 入門

安裝 Rails 和建立第一個應用程式需要知道的一切。

Models

Active Record 基礎知識

Active Record 允許您的 models 與應用程式的資料庫進行互動。本指南將幫助您開始使用 Active Record models 和對資料庫的永續性。

Active Record Migrations

Migrations 是 Active Record 的一個特性，它允許您隨著時間的推移發展您的資料庫架構。遷移允許您使用 Ruby DSL 來描述對錶的更改，而不是在純 SQL 中編寫模式修改。

Active Record 驗證

驗證用於確保僅將有效資料儲存到您的資料庫中。本指南教您如何使用Active Record 的驗證功能在物件進入資料庫之前驗證物件的狀態。

Active Record Callbacks

Callbacks 可以編寫在物件被建立、更新、銷燬等時執行的程式碼。本指南教您如何鉤入 Active Record 物件的這個物件生命週期。

Active Record Associations

在 Active Record 中，關聯是兩個Active Record models 之間的連線。本指南涵蓋了Active Record 提供的所有 associations。

Active Record 查詢介面

與使用原始 SQL 查詢資料庫記錄不同，Active Record 提供了執行相同操作的更好方法。本指南介紹了使用 Active Record 從資料庫中檢索資料的不同方法。

Active Model 基礎知識
Work in progress

Active Model 允許您建立與 Action Pack 整合的普通 Ruby 物件，但不需要 Active Record 來實現資料庫永續性。 Active Model 還有助於構建自定義 ORM，以便在 Rails 框架之外使用。本指南為您提供了開始使用 Active Model 類所需的一切。

Views

Action View 結束 view
Work in progress

Action View 負責為 Web 響應生成 HTML。本指南介紹了Action View。

Rails 中的佈局和渲染

本指南涵蓋了 Action Controller 和 Action View 的基本佈局功能，包括渲染和重定向、使用 content_for 塊以及使用部分。

Action View Helpers
Work in progress

Action View 有 helpers 用於處理從格式化日期和連結到影象到清理和本地化內容的所有內容。本指南介紹了一些更常見的Action View helpers。

Action View 表單 Helpers

由於需要處理表單控制元件命名及其眾多屬性，HTML 表單的編寫和維護很快就會變得乏味。 Rails 透過提供用於生成表單標記的 view helpers 消除了這種複雜性。

Controllers

Action Controller結束view

Action Controllers 是 Rails 中 Web 請求的核心。本指南介紹了 controllers 的工作方式以及它們如何適應您的應用程式的請求週期。它包括sessions、過濾器、cookies、資料流和處理請求引發的異常等主題。

Rails 從外到內路由

Rails 路由器識別 URL 並將它們分派到 controller 的 action。本指南涵蓋了 Rails 路由的面向使用者的功能。如果您想了解如何在您自己的 Rails 應用程式中使用路由，請從這裡開始。

其他元件

Active Support 核心擴充套件

Active Support 提供 Ruby 語言擴充套件和實用程式。它豐富了Ruby 語言，用於 Rails 應用程式的開發，以及Ruby 在Rails 本身的開發。

Action Mailer 基礎知識

本指南為您提供開始從應用程式傳送電子郵件所需的一切，以及 Action Mailer 的許多內部結構。

Action Mailbox 基礎知識
Work in progress

本指南介紹瞭如何使用Action Mailbox 接收電子郵件。

Action Text 結束 view
Work in progress

本指南介紹瞭如何使用Action Text 處理富文字內容。

現役工作基礎

Active Job 是一個用於聲明後臺作業並使它們在各種排隊後端上執行的框架。本指南為您提供了開始建立、排隊和執行後臺作業所需的一切。

Active Storage 結束view

Active Storage 有助於將檔案上傳到雲端儲存服務、轉換上傳和提取元資料。本指南介紹瞭如何將檔案附加到您的 Active Record models。

Action Cable 結束 view

Action Cable 將 WebSocket 與 Rails 應用程式的其餘部分整合在一起。它允許在Ruby 中以與 Rails 應用程式的其餘部分相同的樣式和形式編寫實時功能。本指南介紹了Action Cable 的工作原理，以及如何使用 WebSocket 來建立實時功能。

打包機

本指南將向您展示如何安裝和使用 Webpacker 為 Rails 應用程式的客戶端打包 JavaScript、CSS 和其他資產。

深層發掘

Rails 國際化 (I18n) API

本指南介紹瞭如何為您的應用程式新增國際化。您的應用程式將能夠將內容翻譯成不同的語言、更改複數規則、為每個國家/地區使用正確的日期格式等。
